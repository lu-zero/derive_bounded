## Supported right now

### Struct-style Struct
- [x] Default
- [x] Debug
- [x] Clone
- [x] PartialEq

### Newtype/tuple Struct
- [x] Default
- [x] Debug
- [x] Clone
- [x] PartialEq

### Enums
- [ ] Default
- [x] Debug
- [x] Clone
- [x] PartialEq
